# 🎨 Community 5: Design Proposals — The Drawing Board

> **12 nodes (3 docs + 9 rationale) · Cohesion: 0.20 (moderate — related concepts)**

## 🧍 For humans

**Analogy: The architect's drawing board — napkin sketches, whiteboard sessions, big ideas.**

This is where the **original vision** lives. Before any code was written, someone sat down and designed:
1. **Tree Memory Proposal** — "memory as a hierarchy" (think: folders within folders)
2. **Fractal Memory Proposal** — "memory as a fractal" (think: Russian nesting dolls)
3. **Unified Plan** — "let's do both, they work better together"

Then there's the **advanced theory** — ideas that haven't been implemented yet:
- **Zettelkasten** — the note-taking method that inspired everything
- **Formal Concept Analysis (FCA)** — finding hidden hierarchies in data
- **Sparse Distributed Memory (SDM)** — a mathematical model of human memory
- **Hyperdimensional Computing (HDC/VSA)** — representing concepts as high-dimensional vectors

This community has a moderate cohesion of 0.20. It's more coherent than the Implementation Core (0.14) but less than the HeatTracker (0.28). The ideas here are genuinely related — they all feed into the same vision.
